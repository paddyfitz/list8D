Delete an entry/Nested list

  • To delete an item hover your mouse over the item in question and choose the 'remove from list' icon.
  • You will then be taken to a screen and given the option to remove the item or cancel the instruction.
    • Please note: Once an item is deleted the action is irreversible. When a nested sub list is removed all items attached to it will also be removed. This too is irreversible once deleted.
  • All deleted items will need to be added again if required.
